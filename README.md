# Arduino-Esp8266-switch-emulator
How to use:
  1  Download the code
  2  Open wemos.ino in the Arduino editor.
  3  Change the WI-FI settings.
  4  Define switches and callbacks
  5  Flash
  6  Interact with your simulation
